Saya memiliki pengalaman dalam merancang dan mengimplementasikan berbagai proyek website, mulai dari website galeri seni hingga sistem informasi akademik. Saya sangat tertarik dengan teknologi web modern, . Saya berkomitmen untuk terus belajar dan meningkatkan keterampilan saya dalam menciptakan pengalaman pengguna yang intuitif dan menarik melalui teknologi web.

Selama magang saya, saya mendapat pengalaman praktis dalam pengembangan aplikasi web dan manajemen proyek.
